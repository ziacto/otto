DROP YOUR PDF MANUALS HERE
==========================

Files placed in this folder appear automatically in the app under:
  Knowledge Base → Manuals tab

Supported: .pdf

Suggested files for a BMW 730li 2007 E65 GCC spec:

  - Bentley_E65_E66_Service_Manual.pdf
  - BMW_WDS_Wiring_Diagrams_E65.pdf
  - BMW_TIS_Repair_Procedures.pdf
  - Owners_Manual_730li_GCC.pdf

After adding files, rebuild the app:
  ./gradlew assembleDebug

Tip: rename files with underscores instead of spaces for clean display in the app.
The viewer supports pinch-zoom, pan, and double-tap-to-fit.
